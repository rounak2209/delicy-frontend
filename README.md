# 🍔 DineDash - Modern Food Delivery App

DineDash is a feature-rich, fully responsive food delivery web application inspired by industry giants like Zomato and Swiggy. It offers a premium user interface, advanced search capabilities, and a seamless checkout experience.

## ✨ Key Features

* **🔍 Advanced Deep Search:** Search not just by restaurant name, but deeply through menu categories and specific dish names (e.g., searching "Pizza" shows all restaurants serving pizza).
* **🌓 Dark/Light Mode:** Full support for system-preference or manual dark mode with meticulously crafted Tailwind v4 UI components.
* **🛒 Smart Cart Management:** Advanced cart logic that prevents cross-restaurant ordering, handles dynamic quantity updates, and provides a smooth checkout flow.
* **📜 Order History:** Users can track their past orders with detailed bill breakdowns and delivery statuses.
* **⚡ Premium UX/UI:** * "What's on your mind?" visual category filters.
  * Shimmer UI (Skeleton Loaders) for smooth data fetching transitions.
  * Zomato-style floating menu buttons and custom dropdowns.
  * Dynamic price and veg/non-veg quick filters.
* **🗄️ Smart Mock Backend:** Powered by a customized JSON Server containing realistic restaurant data, diverse menus, and HD contextual images.

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS v4
* **Icons:** Lucide React
* **State Management:** React Context API (`CartContext`, `LocationContext`)
* **Routing:** React Router DOM
* **Backend (Mock):** JSON Server
